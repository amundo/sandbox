Each tag consists of a Category / Feature and a Value separated by a colon <:>. This allows you to sort tags into related groups, as the following examples show.

  • number: singular
  • number: dual
  • grammatical role: A
  • grammatical role: P
  • intonation unit type: final
  • intonation unit type: continuing
  • mystery morphemes: =nk

These tags will be stored so that you can search by tag or tag category later.
